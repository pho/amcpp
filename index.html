This is amcpp

The aim of this project is to give me some entertainment during my boring summer afternoons and looks like I may end up creating a lightweight Ampache client using the Qt framework.

Try it

I'm not distributing any binaries right now, so you will have to

$ git clone git://github.com/pho/amcpp.git
$ cd amcpp
$ qmake
$ make
$ ./amcpp

You will need Qt, qmake and gcc.
